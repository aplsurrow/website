Projects

Financial Statement Web Scrapper

The program shown on the right outputs all the financial statements and metrics for a given public company, allowing for quick and easy access to the financials of a company. Enables analysts to quickly export historical data into an Excel file or other format. Speeds up repetitive tasks when needing to review financial statements promptly.





Monte Carlo Simulation Value at Risk

In this program, we use a Monte Carlo simulation of a given stock portfolio and then use the functions for historical VaR and CVaR to calculate our risk parameters. The main advantage here is we could define individual models/stock dynamics for particular assets. This approach can be useful across many different areas of risk, like margins, volatility, earnings, etc.






Portfolio Optimization Allocation & Effient Frontier Visualization

This program takes in a list of assets and finds the optimal weights for each investment to maximize the Sharpe ratio of the entire portfolio. In reality, it may be wise to include constraints such as; 'must include a minimum % of certain holdings,' 'cannot own more than x % in one investment,' etc.



Ethereum Automated Trading Algorithm

The program uses signals from the simple moving average indicator to buy or sell depending on the conditions set by the trader.
